1. O que é Web Design e qual sua relação com Front-End?

Web Design é o visual do site e como o usuário mexe nele. Front-End transforma esse visual em código que roda no navegador.

2. Cite três princípios de design que melhoram a experiência do usuário.

Clareza, navegação simples e boa legibilidade.

3. Qual a diferença entre Wireframe e Protótipo?

Wireframe é o rascunho estático. Protótipo é o rascunho que dá pra clicar e testar.

4. Liste três ferramentas usadas para criar protótipos.

Figma, Adobe XD e Sketch.

5. Explique a função da tag <header> e dê um exemplo de uso.

Define o topo da página (logo, menu, título).

<header>
    <h1>Meu Site</h1>
</header>

6. Qual a diferença entre uma tag em bloco e uma tag inline?

Bloco ocupa a linha inteira. Inline fica junto do texto, sem quebrar linha.

7. Como aplicar CSS externo em um documento HTML?

No <head>:

<link rel="stylesheet" href="style.css">

8. Cite três propriedades CSS comuns e sua função.

color = cor do texto.
background-color = cor do fundo.
font-size = tamanho da fonte.

9. O que é DOM e como o JavaScript interage com ele?

DOM é a estrutura do HTML viva na memória. O JavaScript muda essa estrutura pra atualizar o site na hora.

10. Escreva um exemplo simples de código JavaScript que exibe uma mensagem no console.

console.log("Olá, mundo!");
